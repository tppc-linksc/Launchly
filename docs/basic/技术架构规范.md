## 1. 技术栈（锁定）

| 层 | 技术 |
| --- | --- |
| Web | Vue 3 + TypeScript + Vite + Element Plus + Pinia + Vue Router + axios |
| API + Worker | NestJS 10 + Prisma ORM + PostgreSQL（单进程，Worker 通过 @nestjs/schedule 内嵌） |
| CLI | TypeScript + commander.js（Self-Host 安装/运维） |
| 部署模板 | Docker Compose（2 服务：postgres + app） |

**禁止**（除非 `docs/work/planning.md` 本地协作计划批准的新任务显式允许）：新语言后端、新前端框架、Redis/Kafka/ES 等额外中间件。

## 6. 版本与配置开关

- **EDITION**：`cloud` | `selfhost`（编译时或运行时），控制 SaaS 独有与 CLI 独有代码路径。
- **BYOS**：MVP 执行路径；Worker **不得**依赖 SaaS 侧挂载宿主 `docker.sock` 作为默认执行模型（Self-Host 亦应以 SSH 到目标为规范执行面）。

---

## 7. 安全与运维要点

- 所有远程命令：**超时、日志、可审计**。
- 用户输入：**class-validator**（NestJS DTO 校验）。
- Schema 变更：**仅通过 Prisma Migrate** 新版本脚本。
- 凭据：**加密 at rest**；日志与 UI **脱敏**。
